# Night Shift and Caffeine. How To Use It Wisely – The Other Shift

*Source*: https://theothershift.com/night-shift-caffeine/

Caffeine, a night shifter's best friend and worst enemy. Caffeine is great at what it does, keeping us alert during our shift, but it can often come back to haunt us when it is time to finally get some sleep. Caffeine itself can be consumed in a number of different forms, but not all caffeine is created equal.
